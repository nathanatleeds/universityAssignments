This project uses flask, first you need to download flask and the virtual environment.
To run it you have to install the relevant files such as SQLAlchemy etc.
Then you do "module add python/3.4.3". Then you type in source flask/bin/activate.
Then do python run.py to run the python file. Then go to the ip address 127.0.0.1:5000.
If you want to see the test files type in python test.py.
